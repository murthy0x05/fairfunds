# Incomplete Tasks

Based on the requirements in the assignment and the current state of the application (`fairfunds.vercel.app`), the following tasks are incomplete. While the backend logic (API actions and services) for these features appears to exist in the codebase, the **frontend user interfaces** are missing from the website:

- [ ] **Manage Group Memberships**: Create a UI to add new members to an existing group, remove members, and manage their temporal membership dates (joined/left dates) as required by the assignment ("Create and manage groups, where membership can change over time").
- [ ] **Create Expenses Manually**: Create an "Add Expense" form/UI. The assignment states "Create and manage expenses", but currently expenses can only be added via the CSV import wizard.
- [ ] **Record Payments / Settle Debts Manually**: Create a UI to manually record settlements or payments between users. Currently, the "Settlements" tab is read-only and displays settlements derived from the CSV, but lacks a "Record Payment" or "Settle Up" action.
